## 简体中文

感谢你使用 Pastel！

我们很高兴带来 Pastel 26.6。

此次更新内容为：

1. 引入全新的 Device GUID（设备 GUID）机制，以避免频繁触发双重认证，参考了 Asspp 的部分原理和实现方式，并同样保存于 macOS Keychain（钥匙串）。
2. 手动下载支持自定义 App ID，并同样支持“不再更新”特性。
3. 修复了其他语言下的部分 UI 显示问题。
4. 优化 macOS Keychain（钥匙串）安全逻辑，减少打扰并提升效率。

26.5.2 更新内容：

1. 支持下载 App 时选择“不再更新”特性。选择后，此 App 将不能再通过 App Store 更新。
2. “下载”板块支持显示版本 ID、是否为可检索更新版本，并优化了部分显示效果。
3. 修复了一些已知问题。

如果在使用 Pastel 时遇到任何问题，请在 GitHub 提交 Issue。

## English

Thank you for using Pastel!

We are happy to bring you Pastel 26.6.

What's new:

1. Introduced a new Device GUID mechanism to avoid frequent two-factor authentication prompts. It references part of Asspp's approach and implementation, and is also stored in the macOS Keychain.
2. Manual downloads now support custom App IDs and the same “No Updates” option.
3. Fixed several UI display issues in non-Chinese languages.
4. Improved macOS Keychain security logic to reduce interruptions and work more efficiently.

Pastel 26.5.2 changes:

1. Added the “No Updates” option when downloading apps. When enabled, the app can no longer be updated through the App Store.
2. The Downloads section now shows version IDs and whether a downloaded version can still be found for updates, with improved visual details.
3. Fixed several known issues.

If you run into any problems while using Pastel, please submit an Issue on GitHub.

## ภาษาไทย

ขอบคุณที่ใช้ Pastel!

เรายินดีที่ได้นำ Pastel 26.6 มาให้ใช้งาน

สิ่งที่อัปเดตในเวอร์ชันนี้:

1. เพิ่มกลไก Device GUID ใหม่เพื่อลดการขอการยืนยันตัวตนแบบสองปัจจัยบ่อยครั้ง โดยอ้างอิงแนวคิดและการใช้งานบางส่วนจาก Asspp และบันทึกไว้ใน macOS Keychain เช่นเดียวกัน
2. การดาวน์โหลดแบบกำหนดเองรองรับ App ID ที่ผู้ใช้ระบุเอง และรองรับตัวเลือก “ไม่อัปเดตอีก” เช่นเดียวกัน
3. แก้ไขปัญหาการแสดงผล UI บางส่วนในภาษาอื่น ๆ
4. ปรับปรุงตรรกะความปลอดภัยของ macOS Keychain เพื่อลดการรบกวนและทำงานได้มีประสิทธิภาพมากขึ้น

สิ่งที่อัปเดตใน Pastel 26.5.2:

1. เพิ่มตัวเลือก “ไม่อัปเดตอีก” เมื่อดาวน์โหลดแอป หากเลือกตัวเลือกนี้ แอปนั้นจะไม่สามารถอัปเดตผ่าน App Store ได้อีก
2. ส่วน “ดาวน์โหลด” รองรับการแสดง Version ID และสถานะว่าเป็นเวอร์ชันที่ค้นหาเพื่อตรวจอัปเดตได้หรือไม่ พร้อมปรับปรุงการแสดงผลบางส่วน
3. แก้ไขปัญหาที่ทราบบางรายการ

หากพบปัญหาระหว่างใช้งาน Pastel โปรดส่ง Issue บน GitHub
